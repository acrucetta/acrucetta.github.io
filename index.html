Hey, I'm Andrés!

I am a software engineer at CoreWeave's cloud platform team, where I help people get access to GPUs to train LLMs and run computer graphics.

I am interested in data platforms, machine learning, and operating systems.

I love tinkering with productivity tooling and have built a set of Rust CLI tools to manage my daily tasks, take notes, and record thoughts from the command line.

Previous Work

  • I was the second engineer at Waltz Health, a Series A health-tech startup focused on making medicines cheaper.
  • I worked on Facebook's product growth team, where we ran A/B tests to improve the platform.
  • I worked at Shelf Engine building machine learning models to forecast grocery volumes and reduce inventory waste.
  • I started CTRL.golf to help golfers improve their swing with machine learning.
